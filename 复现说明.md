# 多帧 DBSCAN 消除深度穿透：复现说明

## 1. 目标
基于技术文档 `DBSCAN/Technical document/Technical document.md` 复现多帧密度聚类去噪流程，实现可运行、可测试、可追溯的独立 Python 程序。

## 2. 问题原理
- 单帧关键点受像素偏移影响，可能从细果梗“滑落”到背景，导致深度穿透（Z 值突变）。
- 多帧观测中，真实果梗点在 3D 空间形成高密度簇；穿透点通常稀疏离散。
- 用 DBSCAN 的密度特性可剔除离群点，再对核心簇做质心融合和收敛判定。

## 3. 技术文档参数到代码的映射
| 技术文档参数 | 数值 | 代码参数 |
| --- | --- | --- |
| 滑窗帧数 N | 15 | `window_size=15` |
| 邻域半径 eps | 15 mm | `eps_mm=15.0` |
| 最小样本 MinPts | 8 | `min_pts=8` |
| 收敛阈值 sigma | 3 mm | `sigma_thresh_mm=3.0` |

收敛判定实现：
- `P_final = mean(C_core)`
- `sigma = sqrt(mean(||p - P_final||^2))`
- `locked = (|C_core| >= MinPts) and (sigma < sigma_thresh_mm)`

## 5. 代码文件与职责
- `DBSCAN/multiframe_dbscan_filter.py`
  - `MultiFrameDBSCANFilter`：滑窗维护 + DBSCAN + 主簇融合 + 锁定判定
  - `FilterResult`：统一输出结构
- `DBSCAN/multi_target_dbscan_tracker.py`
  - `MultiTargetDBSCANTracker`：多目标关联（无稳定ID，最近邻分配）+ 每轨迹独立DBSCAN
  - `TrackState/TrackOutput`：轨迹状态与每帧输出结构
- `DBSCAN/tests/test_multiframe_dbscan_filter.py`
  - TDD 用例（5个）
- `DBSCAN/tests/test_multi_target_dbscan_tracker.py`
  - 多目标TDD用例（3个）
- `DBSCAN/run_demo.py`
  - 合成“主梗簇 + 背景穿透点”并逐帧演示输出
- `DBSCAN/run_multi_target_demo.py`
  - 合成“双果串 + 顺序打乱 + 偶发穿透”并演示多目标稳定输出

## 6. 复现步骤（命令级）
在仓库根目录 `/home/kesl/ke` 下执行：

### 6.1 clone 参考仓库
```bash
git clone https://github.com/chrisjmccormick/dbscan.git DBSCAN/reference/chrisjmccormick-dbscan
git -C DBSCAN/reference/chrisjmccormick-dbscan rev-parse HEAD
```

### 6.2 运行测试
```bash
conda run -n ke pytest DBSCAN/tests -q
```
期望：`8 passed`

### 6.3 运行演示脚本
```bash
conda run -n ke python DBSCAN/run_demo.py
```
期望：前 14 帧 `reason=collecting`，第 15 帧给出 `p_final` 与 `sigma`，并出现 `locked=True`（在当前样例数据下）。

多目标演示（无稳定ID）：
```bash
conda run -n ke python DBSCAN/run_multi_target_demo.py
```
期望：在检测顺序打乱、偶发深度穿透情况下，仍保持2条稳定轨迹并最终锁定。

### 6.4 生成可视化图片
```bash
conda run -n ke python DBSCAN/visualize_demo.py
```
默认输出：
- `DBSCAN/output/dbscan_multiframe_demo.png`

图像内容：
- 左图：3D 散点（绿色核心簇、红色穿透噪声、黑色 `P_final`）
- 右图：逐帧 `sigma` 收敛曲线与阈值线（`3mm`）

按 `cluster` 风格读取 txt：
```bash
conda run -n ke python DBSCAN/visualize_demo.py \
  --txt DBSCAN/output/sample_points.txt \
  --output DBSCAN/output/dbscan_from_txt_cluster_style.png
```
说明：
- `--txt` 支持行内格式如 `[x, y, z]`
- 图中会显示红色中心点、红色包围框、黑色框内点，并打印 `in_box` 统计

多目标可视化（无稳定ID）：
```bash
conda run -n ke python DBSCAN/visualize_multi_target_demo.py
```
默认输出：
- `DBSCAN/output/multi_target_tracker_demo.png`

## 7. 验证要点
- 滑窗未满时不允许锁定
- 背景穿透点被当作噪声剔除，不影响主簇质心
- 只有在核心簇点数和 sigma 同时满足阈值时才锁定

## 8. 可扩展建议
- 将 `add_observation` 接口接入实时视觉管线（每帧输入世界坐标）
- 针对不同机型/相机标定误差调参 `eps_mm` 与 `sigma_thresh_mm`
- 若后续需要高吞吐，可将 `_region_query` 改为 KDTree 加速
